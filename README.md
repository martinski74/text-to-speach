# Text-to-Speech Vue 3 App

A modern, responsive web application built with **Vue 3** and **Vite** that converts text into synthesized speech using the native **Web Speech API**.

## ✨ Features

- 🎙️ **Speech Synthesis**: High-quality text-to-speech conversion.
- 🌍 **Multi-language Support**: Automatically detects and filters available system voices.
- ⚙️ **Fine-grained Controls**: Adjust speed (rate), pitch, and volume.
- 📊 **Visual Feedback**: Real-time CSS waveform animation and status indicators.
- 📱 **Responsive Design**: Optimized for desktop and mobile devices.
- 🌑 **Dark Mode UI**: Clean, modern aesthetic with a "glassmorphism" feel.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone or download the repository.
2. Navigate to the project directory:
   ```bash
   cd text-to-speech
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

To start the local development server:
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:5173`.

### Production

To build the project for production:
```bash
npm run build
```
The output will be in the `dist/` directory.

## 🛠️ Built With

- **Vue 3** (Composition API)
- **Vite** (Build Tool)
- **Web Speech API** (Native Browser API)
